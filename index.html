Team Profile & Stats ✅

Team details (name, city, logo, founded date) and all official stats (matches, wins, losses, win %, toss, bat/field first) are fully available from the CricHeroes API.

Match-level Data ✅

Full match history (8 past matches) is loaded from the CricHeroes matches API — opponent, venue, scores, result, toss details, and tournament. Data resets automatically to the live CricHeroes dataset on first load.

Members / Squad ✅

All 30 squad members are fetched from the CricHeroes members tab API — including names, roles, profile photos, and captain/admin/PRO status.

Photos ✅

Match photos are fetched from the api.cricheroes.in/get-team-match-photo REST API and auto-populated into the gallery section.

Leaderboard ❌

Individual player leaderboard stats (top scorers, wicket-takers) are PRO-gated on CricHeroes. Upgrade the team account to access individual player performance breakdowns via the API.

Filters & Tournaments ✅

Tournament list, overs options (25, 30), ball type (Leather), match formats, and tournament categories (Corporate, Open) are all available and displayed above.
